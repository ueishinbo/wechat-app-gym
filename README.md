# wechat-app-gym

微信小程序体育馆预约系统demo
## 补充说明
支付页面：支付密码填任意字符即可。
订单页面：没有做我的票价UI，故如果在支付页面支付成功后并不会在我的订单查找到，可在首页点击查看我的二维码，此页面代表下一场即将开始的订单。
